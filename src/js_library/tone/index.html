🎵 Tone.js Integration

Tone.js is a powerful Web Audio framework for creating interactive music in the browser. Our integration provides a comprehensive wrapper with synthesizers, effects, drum machines, and more!

🎼 Available Features

  • Multiple Synthesizer Types (AM, FM, Mono, Poly, etc.)
  • Audio File Playback & Management
  • Drum Machine with Pattern Sequencing
  • Audio Effects (Reverb, Delay, Distortion, etc.)
  • Sequence & Pattern Creation
  • Transport Controls (Start, Stop, Tempo)
  • Squirrel Framework Integration
  • Visual Audio Analysis & Display
  • Real-time Audio Processing
  • MIDI Integration Support
  • Recording & Export Capabilities
  • Interactive Keyboard & Piano
