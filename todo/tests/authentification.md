# Authentification et accès invité

- [ ] [Fonctionnel][Compte] Créer un compte.
- [ ] [Fonctionnel][Invité] Accéder sans créer de compte.
- [ ] [Fonctionnel][Compte] Cliquer sur créer un compte.
- [ ] [Fonctionnel][Téléphone] Cliquer sur l’icône du bas sans renseigner le numéro de téléphone.
- [ ] [Fonctionnel][Navigation] Vérifier le retour à la racine.
- [ ] [Fonctionnel][Compte] Recliquer sur créer un compte.
- [ ] [Fonctionnel][Téléphone] Renseigner le numéro de téléphone.
- [ ] [Fonctionnel][Téléphone] Valider le numéro de téléphone.
- [ ] [Fonctionnel][OTP] Vérifier le passage à l’écran OTP.
- [ ] [Fonctionnel][OTP] Vérifier la réception de l’OTP.
- [ ] [Fonctionnel][OTP] Retourner à la racine si l’OTP n’est pas disponible.
- [ ] [Fonctionnel][OTP] Renseigner l’OTP.
- [ ] [Fonctionnel][OTP] Valider l’OTP.
- [ ] [Fonctionnel][Mot de passe] Vérifier le passage à l’écran de renseignement du mot de passe.
- [ ] [Fonctionnel][Mot de passe] Renseigner le mot de passe.
- [ ] [Fonctionnel][Mot de passe] Valider le mot de passe.
- [ ] [Fonctionnel][Session] Vérifier que l’environnement se connecte correctement.
